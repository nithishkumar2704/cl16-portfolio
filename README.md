## üèÅ Overview
This project is a high-fidelity **Personal Portfolio Website** designed for F1 Driver Charles Leclerc.  
It moves away from traditional "image-heavy" sports sites and embraces a **Typographic & Data-Driven** aesthetic (Swiss Style).

**The Vibe:** Dark, Expensive, Technical.  
**The Goal:** To create a digital experience that feels like a high-performance dashboard.
